================================================================================
  7. NOTES TECHNIQUES
================================================================================

  Module Go : github.com/yoremi/rldev-go/rlc  (et /kprl, /common)
  Go version : 1.22+
  Nommage Go : package "goto" → "gotojmp", package "select" → "sel"

  Fixes rldev2026 portés en Go :
  - ini.go : clés 5 niveaux IDENT.NNN.NNN.TEXT.NNN (Tomoyo After Steam)
  - ini.go : clés 5 niveaux IDENT.NNN.TEXT.TEXT.TEXT (Clannad Side Stories)
  - kfn.go : iniParser 5-level keys
  - codegen.go : OBJBACKCHILDMOJI fix

  Architecture Go vs OCaml :
  - OCaml utilise des refs mutables globales (Global.expr__normalise_and_get_const)
    → Go utilise des callbacks dans meta.State.CompileStatements
  - OCaml utilise des variant types polymorphes (`Int | `Str | `Store ...)
    → Go utilise des interfaces (ast.Expr, ast.Stmt) + type switch
  - OCaml utilise des DynArray mutables pour les token streams
    → Go utilise des slices
